## üí° Pro Tips for Keyword Optimization

### On-Page SEO Checklist (Every Blog Post)

- [ ] **Title:** Includes primary keyword (within 60 characters)
- [ ] **Meta Description:** Includes keyword + CTA (155-160 chars)
- [ ] **H1:** Keyword variant (only 1 per page)
- [ ] **H2-H4:** Secondary keywords and related terms
- [ ] **First 100 words:** Natural keyword mention (3-4 times max)
- [ ] **Internal Links:** 8-10 to related pages (target keyword variants)
- [ ] **Images:** Alt text includes keywords (e.g., "cyberpunk midjourney prompt example")
- [ ] **URL:** Slug format: `/blog/[keyword-variant]/`
- [ ] **Schema Markup:** Article or BlogPosting schema
- [ ] **Mobile Test:** Responsive design, fast loading

### Keyword Placement Best Practices

**DO:**
- Natural mention in first paragraph
- Title and meta description
- H1 and H2 tags
- Image alt text
- Internal link anchor text
- First and last 100 words

**DON'T:**
- Keyword stuffing (keep it under 1.5% density)
- Forcing keywords unnaturally
- Repeating exact keyword in every sentence
- Using keyword in URL if unnatural
- Ignoring related keywords (use variations)

### Related Keywords to Include in Every Post

When writing a post about "Cyberpunk Midjourney prompts," naturally include:
- "Neon prompts"
- "Futuristic Midjourney"
- "Dystopian city prompts"
- "Midjourney sci-fi"
- "Dark cyberpunk aesthetic"
- "AI cyberpunk art"

**Why:** Google understands semantic relationships. Including related terms improves relevance.

## üìû Questions to Ask Before Writing

Before starting ANY content, ask:

1. **What keyword am I targeting?** (Should have 1 primary + 3-5 secondary)
2. **What is the search intent?** (Learn? Find? Buy? Do?)
3. **How long should it be?** (Tier 1: 800-1200 | Tier 2: 1500-2000 | Tier 3: 1500-2000)
4. **Who am I competing against?** (Check Google SERP for keyword)
5. **What will rank better?** (More helpful? More detailed? Better examples?)
6. **Can I link internally?** (To category pages? Tag pages? Other blogs?)
7. **What's my CTA?** (Browse prompts? Upgrade? Explore more?)
8. **Will this convert?** (Traffic ‚Üí Users ‚Üí Conversions?)
